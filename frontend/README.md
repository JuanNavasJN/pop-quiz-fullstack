# pop-quiz-frontend

## Description

Frontend side of the Pop Quiz. Made using Next.js, TypeScript and Material UI.

## Installation

```bash
$ npm install
```

## Required environment variables

```bash
NEXT_PUBLIC_BACKEND_HOST=
```

## Running the app

```bash
# development
$ npm run dev

# build production mode
$ npm run build

```
