# pop-quiz-backend

## Description

Backend side of the Pop Quiz. Made using [Nest](https://github.com/nestjs/nest) framework TypeScript along with MongoDB.

## Installation

```bash
$ npm install
```

## Required environment variables

```bash
PORT=

MONGODB_URI=

MAIL_HOST=
MAIL_PORT=
MAIL_USER=
MAIL_PASSWORD=
MAIL_FROM=

FRONTEND_HOST=

NEWS_API_KEY=
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod


```
